Sobre Mim

Olá, sou a Akemi!

Estagiária no Itaú Unibanco, atuando na estruturação e automatização de processos e bases de dados alinhadas às regras de negócio da área de Planejamento Comercial de Veículos, contribuindo para maior eficiência e agilidade na disponibilização das informações. Entre os próximos aprendizados, tenho grande expectativa em aprofundar meus conhecimentos em Deep Learning, Engenharia de Dados, Backend e Cloud, temas que enxergo como estratégicos para desenvolver soluções cada vez mais robustas, escaláveis e alinhadas às demandas do mercado.

Projeto API

Desenvolvimento de API REST em FastAPI, com CRUD completo e integração a banco de dados relacional. Modelagem de dados em PostgreSQL utilizando SQLAlchemy ORM, com definição de chaves e relacionamentos. ExpressLine

"ExpressLine" é um sistema para modelar conexões diretas entre cidades com Listas Duplamente Encadeadas (LDE) genéricas. Permite cadastrar cidades e ligações, calcular tempo de entrega e buscar rotas por tempo limite, com interface gráfica e API interna para manipulação e consulta dos dados.

Repositório no GitHub

  • Java
  • JOptionPane
  • LDE
  • API própria

Controle Hotel Pet

"Controle Hotel Pet" é um sistema desenvolvido para gerenciar as operações de um hotel para pets, abrangendo funcionalidades como cadastro, edição, consulta, listagem e hospedagem de animais, com integração a um banco de dados relacional.

Repositório no GitHub

  • Python
  • SQLAlchemy
  • MySQL
